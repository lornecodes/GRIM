# Graph Studio — Agent Topology Visualizer

**Date**: 2026-03-03
**Version**: 0.0.7
**Scope**: core, server, ui, tests

## Overview

New "Graph Studio" tab on the Agent Team page that visualizes the full GRIM LangGraph topology as an interactive DAG. Shows live execution overlay during conversations, node inspection with tool/config details, agent toggle controls, and active session tracking.

## Backend

- **`core/graph_topology.py`** — Static topology data module: 11 infrastructure node metadata entries (with descriptions, colors, routing rules), 19 edges with types/labels, fixed (col, row) DAG layout positions for all 15 nodes
- **`GET /api/graph/topology`** — Merges static infra metadata + GRAPH_NODE_METADATA (companions) + dynamic agent registry into a complete topology response with layout positions, enabled state, and node types
- **`GET /api/graph/sessions`** — Returns count and IDs of active WebSocket sessions
- **`_active_ws_sessions`** — Module-level set in `app.py` tracks live WS connections (add on accept, discard on disconnect/finally)

## Frontend

- **`lib/graphTopology.ts`** — TypeScript types (TopologyNode, TopologyEdge, GraphTopology, NodeType, EdgeType) and `toCanvasPos()` layout helper
- **`hooks/useGraphOverlay.ts`** — Derives live execution state from Zustand store traces: active/completed nodes, traversed edges, duration badges
- **`components/graph/GraphStatusBar.tsx`** — Status strip: active session count, node/edge counts, pulsing "Executing" indicator
- **`components/graph/NodeInspector.tsx`** — Side panel: node name, type badge, tier badge, execution state (active/completed/idle with duration), description, routing rules for router nodes, collapsible tools list, agent toggle switch for toggleable agents, "always on" badge
- **`components/graph/TopologyGraph.tsx`** — Canvas renderer using `react-force-graph-2d` with all d3 forces disabled, positions pinned via `fx`/`fy`. Custom `nodeCanvasObject` with radial glow for active nodes, pulse animation via rAF, completion rings, duration badges. Custom `linkCanvasObject` with dashed conditional edges, solid static edges, arrow heads, edge labels, quadratic bezier arc for re_dispatch loop
- **`components/graph/GraphStudio.tsx`** — Container: fetches topology on mount, polls sessions every 5s, ResizeObserver for responsive canvas, split layout with inspector panel, agent toggle handler with optimistic update, color legend
- **`pages/AgentTeam.tsx`** — Added underline-style tab bar ("Team" / "Graph Studio") with lazy-loaded Graph Studio via React.lazy + Suspense
