# v0.10 Core Loop Reshape

**Date**: 2026-03-04
**Tag**: v0.0.10.0
**Stories**: grim-006-010 through grim-006-014

## Summary

Major architectural milestone: GRIM's graph evolves from keyword-based linear routing to an LLM-backed cyclic loop with structured output classification. All behind `config.use_companion_router=True` — v0.0.6 keyword routing remains the default.

## What Changed

### New Components
- **Intent Classifier** (`core/nodes/intent_classifier.py`): 3-tier classification — skill hint hard override → LLM structured output via `with_structured_output(RoutingDecision)` → keyword fallback safety net
- **Companion Router** (`core/nodes/companion_router.py`): Single routing node replacing graph_router + router two-stage chain. Produces graph_target, mode, delegation_type, selected_model
- **Response Generator** (`core/nodes/response_generator.py`): Loop heartbeat — processes SubgraphOutput, updates objectives, decides continue/exit with safety valve (max_loops)
- **Subgraph Wrappers** (`core/subgraphs/`): 4 adapter nodes wrapping existing logic into SubgraphOutput interface
  - `conversation.py`: companion + personal_companion (routes by graph_target)
  - `planning.py`: planning_companion with execution intent detection
  - `research.py`: dispatch wrapper for research/codebase/memory agents
  - `code.py`: dispatch wrapper for ironclaw with auto_continue detection

### State & Models
- **GRIMState v0.10 schema**: `schema_version`, objectives (with reducer), subgraph_output, loop control fields, context_stack, subgraph_history, continuation_intent, trace_id
- **Objective system**: Pydantic model with lifecycle (PENDING → ACTIVE → COMPLETE/BLOCKED/FAILED), parent/children hierarchy, auto_continue flag, target_subgraph
- **RoutingDecision**: Typed structured output model for LLM classification
- **SubgraphOutput**: Common interface — response, artifacts, memory_updates, objective_updates, continuation, source_subgraph

### Graph Topology (v0.10)
```
identity → compress → memory → skill_match → companion_router →
  [conversation | planning | research | code] →
  response_generator → [continue: companion_router | exit: integrate] →
  integrate → evolve → END
```
